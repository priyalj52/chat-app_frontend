## MERN stack chat application  with dark/light mode
### User can perform following functions:
<ul>
  <li>
    Sign UP
  </li>
   <li>
    Login
  </li>
   <li>
    Real time chatting 
    <ul>
      <li>See Notifications </li>
      <li>mark notifications read/unread </li>
       <li>See which users are online </li>
    </ul>
  </li>
 
 
  
</ul>
